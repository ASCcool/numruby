# nmatrix_reloaded
Reimplementing NMatrix for fun

## NMatrix

Fast Numerical Linear Algebra Library for Ruby

## Installation

```sh
git clone https://github.com/prasunanand/nmatrix_reloaded
cd nmatrix_reloaded/
gem install bundler
bundle install
rake compile
```

Run the tests using

```sh
rake test
```

If you want to try out the code without installing:

```sh
rake pry
```
